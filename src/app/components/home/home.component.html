Java Coding Challenge

As anAPI user
I want to submit a batch of animal records to a RESTful endpoint
So I can update my animal maps in real-time

Acceptance Criteria

  • Provide a single endpoint (/animals) built with the Spring Boot framework
  • It should accept a POST request via the attached Postman collection
  • In addition, the endpoint should update a Google Firebase REST endpoint via a PATCH request, supplying the same data
  • Activity to be logged to a text file

Enhancements (optional)

  • Validation of request data
  • Javadocs

Data

  • The existing Firebase dataset is visible at https://eshepherd-challenge-default-rtdb.asia-southeast1.firebasedatabase.app/.json?print=pretty.
  • The attached Postman Collection has commands and global variables already setup, no need for you to edit anything.
  • Import the collection and then start by initializing a new dataset for yourself via the “Firebase API / create animals” command.
    This will save a new "{{"name"}}" for you as a global variable, such that all subsequent commands will work.
  • The other commands in the “Firebase API” folder are there if you need them, they use the established global variables
  • GET https://"{{"database"}}"/"{{"name"}}"".json           // get "animals" 
PUT https://"{{"database"}}"/"{{"name"}}"".json           // "reset animals"
  • You can test your own endpoint via “Your API”.
  • More information: Firebase Realtime Database API Docs
